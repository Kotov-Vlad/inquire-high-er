Для работы кода нужно установить node(для каждой системы в интернете есть информация). Дальше вводите в папке с проектом npm install yarn, далее чтобы установить все зависимости введите yarn add, далее чтобы скомпилировать typescript в js введите yarn run build(это вводите при изменение любых исходных файлов). Все настройки в ./src/index.ts файле. Если прокси не нужен то удалите httpAgent из export const client = getTwitterClient, если нужен то в в объекте config найдите credentials и впишите "username:proxy" вашего прокси. hostUrl - URL вашего хоста вместе с протоколом https://. TelegramChatId - id вашего телеграм чата, например  @bzbro. MaximumExcessPer это процент превышения, как вы просили он равен 0.2, maxCheckPosts - количество раз проверки поста на превышение. numTweetsInitEr это количество постов, с которых нужно считать первоначальный ER. Остальное можно не трогать. Для запуска yarn run start
